2016 年度计划

All children, except one, grow up. — J. M. Barrie，《The Adventures of Peter Pan》

一、学校·毕业

如果把生活看做一本书，我猜2016年应该属于承前启后的 Chapter2，沉默和信息量大，上半年读最最最重要的事情就是顺利毕业

  • 毕设开题，导师，项目和论文上的事情,作别 HUST
  • 实习成绩单，毕设接收书，三方，文件和流程上的东西

二、生活·旅人

虽然没看完《断捨離》，但对理念还是和赞同的，“断绝不需要的东西；舍去多余的事物；脱离对物品的执着”。
感觉这个标题还是很契合我当前阶段的，当下不杂，既往不恋，未来不迎。尽力摒弃以往不好的，为了适应工作和生活做出一些必要的调整，达到更好的生活状态。

  • 3月房租到期，找一个带厨房的新狗窝，要做史上最年轻的特级厨师
  • 最好有篮球场的小区，每周末坚持蹦跶蹦跶
  • 烫个卷毛，省的每天洗剪吹啦
  • 买一台 PS4，一些想玩的游戏
  • 毕业旅行，来一次中二脱宅之旅 — 东京、箱根 、秋叶原、新宿、镰仓
  • 晚上加班，坚持去健身房锻炼
  • 最高目标，吃早餐，不熬夜！有一个不同于大学的健康生物钟
  • 逐渐脱离动漫和网文，转向 BBC 纪录片和 kindle库存，多读英文

三、工作

工作上的事情也不好多说，暂时在适应和摸索期，目标在毕业之前完成角色转变，找到专注的方向和成熟的节奏

  • 一些感兴趣的方向
    • IM业务SDK，重构（KPI）
    • Node（深入）
    • 组件库（贡献度）
    • react，isomorphic， redux，koa，无线基础
    • React Native（ app 容器）
  • 眼界保持宽广，关注开源，保持每周末都有github commit，坚持自己的产出
    • yillia：拿自己的VPS去跑node代理，提供instagram图片同步服务，帮忙修复yilia和hexo的一些问题
    • gitlab-tree: 支持新版gitlab-ui的改版
    • chrome 插件（公司内部的和外部的几个idea）
    • 网盘搜索和下载的mac app
    • 学学swift，产出自己的第一个IOS APP
    • 做一个android app， JAVA or RN
    • 网易云和豆瓣fm 导入，or 爬虫
  • 产品，项目管理，移动端
